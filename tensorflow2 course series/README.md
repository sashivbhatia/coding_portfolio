# UCL TensorFlow 2 Capstone Projects  

This repository contains my two capstone projects completed as part of the **Learning TensorFlow 2 Course Series** on Coursera, offered by **University College London (UCL)** and taught by **Dr. Kevin Webster**.  

Both projects are peer-assessed and demonstrate the application of TensorFlow 2 concepts to real-world machine learning problems.  

---

## 📂 Projects  

### 1. Image Classifier for the SVHN Dataset  
- **Objective:** Build, train, validate, and save a neural network that classifies real-world images of digits from the **Street View House Numbers (SVHN) dataset**.  
- **Key Concepts:**  
  - Data preprocessing and augmentation  
  - Neural network model design in TensorFlow 2  
  - Training, validation, and testing workflows  
  - Model saving and loading for reuse  

---

### 2. Neural Translation Model (English → German)  
- **Objective:** Develop a neural sequence-to-sequence translation model that translates sentences from **English to German**.  
- **Key Concepts:**  
  - Data preprocessing and tokenization for NLP tasks  
  - Sequence modeling with RNNs/GRUs/LSTMs  
  - Freezing and fine-tuning layers  
  - Flexible model architectures in TensorFlow 2
